# QA Skill

## Purpose

Describe how future quality assurance work should validate generated projects before handoff.

## Responsibilities

- Compare generated output against the specification
- Check completeness, consistency, and quality
- Identify defects, omissions, and regressions
- Provide approval or rework guidance

## Inputs

- Project specification
- Generated project
- Planner and agent outputs

## Outputs

- QA findings
- Validation summary
- Rework recommendations
